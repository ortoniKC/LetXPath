# Ortoni Studio: The Ultimate DevTools Workspace for Automation Engineers 🚀

Say goodbye to manual locator writing and flaky tests. Welcome to **Ortoni Studio** (formerly LetXPath) — the most powerful, feature-rich Chrome DevTools extension built to supercharge your test automation workflow.

Whether you're using **Playwright, Selenium, Cypress, or your own in-house framework**, Ortoni Studio generates reliable, optimized locators in a single click, saving you hours of writing boilerplate code.

---

## ⚡ Why Choose Ortoni Studio? (The Selling Points)

Traditional inspector tools give you long, fragile XPaths that break the moment your developer changes a class name. Ortoni Studio is engineered differently:

* **🎯 1-Click Smart Locators**: Hover, click, and instantly get the cleanest, most reliable XPath, CSS, or native Playwright locator (`getByRole`, `getByText`, etc.) for any element on the page.
* **🛡️ Bulletproof Test Stability**: Prioritize selector attributes (like `data-testid` or `data-cy`) so your test suites stay green even when the application UI changes.
* **✍️ Custom POM Templates Builder**: Do you have a custom in-house automation framework? Define your own syntax rules (Click, SendKeys, GetText) and watch Ortoni Studio auto-compile custom code snippets for you on the fly.
* **📼 No-Code Smart Recorder**: Record your manual actions (clicks, text input, dropdown selections) and instantly export a fully structured Playwright or Cypress test script.
* **🔍 Live Locator Verifier**: Test, highlight, and verify selector paths in real-time directly on the page. No console commands required.
* **🎓 The ONLY Tool That Teaches You**: Built by automation developers, for automation developers. We provide step-by-step video tutorials showing you exactly how this tool was built!

## 📖 How to Use Ortoni Studio (Get Started in 30 Seconds)

1. **Install the Extension**: Add Ortoni Studio to your Chrome browser.
2. **Open DevTools**: Navigate to any web page, press `F12` (or right-click and choose **Inspect**), and click on the **Ortoni Studio** panel.
3. **Inspect Elements**: Hover and click on any button, input field, or element on your page. Instantly see verified XPaths, CSS, and Playwright locators.
4. **Copy & Run**: Choose your framework target, click to copy the generated snippet, and paste it straight into your IDE.
5. **Tailor Your Workspace**: Click the Settings gear to configure your attribute priority order or construct your custom POM templates.
