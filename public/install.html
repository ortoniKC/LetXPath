Welcome to Ortoni Studio

Thank you for installing! Ortoni Studio is now active. Follow this quick guide to start finding and testing locators like a pro.

⚠️
Restart Recommended
Please restart your browser or reload your currently open tabs to enable Ortoni Studio's DOM listeners on those pages.
Step 1

Open Ortoni Studio

  • Navigate to any webpage you want to automate.
  • Open DevTools (Press F12 or right-click and choose Inspect).
  • Select the Ortoni Studio tab located in the DevTools sidebar.
Step 2

Inspect Elements

  • Click on the inspect tool icon in DevTools.
  • Hover over and click any element on the page.
  • Ortoni Studio immediately locks the element and displays its details.
Step 3

Copy Locators

  • Click directly on any generated locator expression in the panel.
  • The selector (XPath, CSS, etc.) is copied to your clipboard instantly.
  • Click the Find button to verify and highlight elements.
Step 4

Framework Snippets

  • Click the action dropdown next to the locator.
  • Select an action snippet (e.g. click, sendKeys, etc.).
  • Get fully formatted driver code statements for Selenium or Playwright.
Step 5

Axes XPath Relations

  • Right-click the parent/source element, then the child/target element on the page.
  • Choose relative combinations from the Axes tab in Ortoni Studio.
  • Easily automate elements with dynamic properties.
Step 6

Universal Evaluator

  • Go to the Tools tab in the Ortoni Studio panel.
  • Type or paste any XPath, CSS, or Playwright locator string.
  • Instantly see the number of matches and highlight them on the DOM.
Step 7

Manage & Settings

  • Click on the settings/extension icon on your Chrome toolbar.
  • Update your default code snippet targets and custom templates.
  • Set up variables naming patterns for clean script exports.
