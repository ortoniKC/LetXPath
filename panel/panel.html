Simple, Useful, Open-Source XPath Finder

LetXPath by LetCode with Koushik is an open-source tool for finding XPath and CSS selectors with a single click, offering code snippets based on the element type.

    Features:
  • Single-click XPath generation
  • Dynamic and Axes-based XPaths
  • Code snippets for Selenium (Java, Python, C#), Protractor JS, Playwright Node & Java
